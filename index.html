HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membangun struktur dan konten halaman web, seperti teks, gambar, video, dan tautan. HTML bekerja dengan elemen dan tag untuk menentukan bagaimana informasi ditampilkan di browser. lihat contohnya

CSS

CSS (Cascading Style Sheets) adalah bahasa desain untuk mengatur tampilan halaman web. CSS digunakan untuk mengubah warna, font, ukuran, dan tata letak elemen HTML. Dengan CSS, tampilan web bisa dibuat lebih menarik dan responsif. CSS bekerja dengan selektor dan properti untuk menentukan gaya pada elemen tertentu. lihat contohnya

JAVASCRIPT

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif dan dinamis. Dengan JavaScript, elemen pada halaman dapat diubah tanpa perlu memuat ulang, seperti animasi, validasi formulir, atau efek transisi. lihat contohnya

FRONTEND DEVELOPER

Frontend development adalah proses membangun tampilan dan interaksi pada website atau aplikasi web menggunakan HTML, CSS, dan JavaScript. Tujuannya adalah membuat antarmuka yang menarik, responsif, dan mudah digunakan.

BACKEND DEVELOPER

Backend development adalah proses pengembangan bagian server dari sebuah website atau aplikasi yang mengelola database, autentikasi, dan logika bisnis. Backend bekerja dengan bahasa seperti Node.js, Python, atau PHP serta menggunakan database seperti MySQL atau MongoDB untuk menyimpan dan mengelola data.

FULLSTACK DEVELOPER

Fullstack development adalah proses pengembangan web yang mencakup baik frontend (tampilan pengguna) maupun backend (server, database, dan logika bisnis). Seorang fullstack developer mampu bekerja dengan teknologi seperti HTML, CSS, JavaScript untuk frontend serta Node.js, Python, atau PHP untuk backend.
